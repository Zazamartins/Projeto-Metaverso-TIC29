## Aviso de Limitação Técnica (Hardware)
Este repositório contém a documentação e o planejamento técnico estrutural do projeto final. 

Atualmente, meu setup de desenvolvimento não dispõe de memória RAM e recursos de processamento gráfico (GPU) suficientes para executar o Unity Editor simultaneamente com o Meta XR SDK de forma estável. Essa limitação de hardware inviabilizou a compilação, o *baking* de iluminação e os testes do ambiente 3D em tempo real sem travamentos críticos do sistema.

## Entrega Avaliativa (Relatório Técnico)
Para cumprir os requisitos estruturais e teóricos da avaliação, o planejamento completo do projeto foi documentado em formato de Relatório Técnico. O documento abrange:
